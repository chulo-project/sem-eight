## ğŸ’¡ Top Exam Tips for AD

1.  **Diagrams are Crucial:** Practice drawing EER diagrams (Standard Crow's foot notation) and Query Trees.
2.  **Query Optimization:** Understand the heuristic rules (pushing selection/projection down) thoroughly. You will likely get a practical problem on this.
3.  **NoSQL & Big Data:** This is a modern topic. Be clear on CAP Theorem, MapReduce phases, and the difference between Document/Key-Value/Column stores.
4.  **Algorithms:** Know the steps for Mapping EER to Relational, and the Query Processing phases.
5.  **Comparison:** Be ready to compare RDBMS vs ODBMS vs ORDBMS.
