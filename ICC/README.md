## ðŸ’¡ Top Exam Tips for ICC

1.  **NIST Definition:** Memorize the 5 Characteristics, 3 Service Models, and 4 Deployment Models (5-3-2 principle).
2.  **Architecture Diagrams:** Practice drawing the Cloud Reference Architecture, Aneka Architecture, and Hypervisor Types (Type 1 vs Type 2).
3.  **MapReduce:** Understand the workflow (Input Split -> Map -> Shuffle -> Reduce). Be able to write a simple word count pseudo-code/explanation.
4.  **Security:** Focus on the "Shared Responsibility Model". Know who secures what in IaaS, PaaS, and SaaS.
5.  **Virtualization:** Be clear on the levels of virtualization and the role of the Hypervisor.
