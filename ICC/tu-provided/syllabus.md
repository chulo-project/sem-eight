**Course Title:** Introduction to Cloud Computing
**Course No:** CSC467
**Semester:** VIII
**Nature of Course:** Theory + Lab

- **Full Marks:** 60 + 20 + 20
- **Pass Marks:** 24 + 8 + 8
- **Credit Hours:** 3

_Course Description:_ This course covers concepts of cloud computing including introduction, architectures, virtualization, programming models, security, and platforms and applications.

_Course Objective:_ Provide theoretical and practical knowledge of cloud computing, including designing, implementing, and managing cloud systems.

---

### Course Contents

**Unit 1 – Introduction to Cloud Computing** (6 hrs)

- Evolution, characteristics, types of cloud services, benefits, challenges, applications, storage, requirements, adoption.

**Unit 2 – Cloud Computing Architecture** (6 hrs)

- PaaS, SaaS, IaaS, public/private/community/hybrid clouds, SOA‑based design, security, trust, privacy.

**Unit 3 – Cloud Virtualization Technology** (10 hrs)

- Virtualization basics, types, implementation levels, benefits, server virtualization, hypervisors, load balancing, infrastructure needs.

**Unit 4 – Cloud Programming Models** (12 hrs)

- Thread, task, Map‑Reduce programming, parallel efficiency, enterprise batch processing, comparisons.

**Unit 5 – Cloud Security** (6 hrs)

- Security issues, challenges, risks, SaaS security, monitoring, architecture design, data/application security, VM security, legal aspects, multi‑tenancy.

**Unit 6 – Cloud Platforms and Applications** (12 hrs)

- Web services, AppEngine, Azure, Aneka, open challenges, scientific, business, consumer applications.
